Como funciona o sorteador de números

O sorteador utiliza um algoritmo de geração aleatória para cirar números dentro do intervalo especificado pelo usuário

Posso escolher o intervalo dos números ?

Sim, você poide definir os valores mínimo e máximo para o intervalo dos números sorteados
